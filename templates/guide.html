📖 使用指南

🚀 快速开始

我们提供两种参赛方式，任选其一即可：

推荐

方式一：bg_tool 独立工具

不需要安装 HDT，一个独立的桌面小工具就能参赛。

  1. 下载 bg_tool 压缩包，解压到任意目录
  2. 先启动炉石传说，进入主界面
  3. 右键 bg_tool.exe → 以管理员身份运行（避免 HearthMirror 读取失败）
  4. 工具自动连接服务器，界面上直接显示验证码 → 点击复制
  5. 用验证码在网站注册/登录，然后报名参赛

📌 支持断线重连检测、最近战绩统计、验证码一键复制

📌 需要 Windows 系统，开箱即用，不需要额外配置

⚠️ 如果日志显示「HearthMirror 读取失败: 拒绝访问」，请右键 bg_tool.exe → 以管理员身份运行（已验证有效）

方式二：HDT 插件

如果你已经在用 HDT（Hearthstone Deck Tracker），可以装插件。

  1. 安装 HDT
  2. 下载插件 HDT_BGTracker.dll，放到 HDT 插件目录
  3. 重启 HDT → 在「插件」页面确认 BGTracker 已启用
  4. 进入炉石主界面 → 插件自动连接服务器，验证码出现在插件设置页
  5. 用验证码在网站注册/登录，然后报名参赛

📌 插件目录通常在 %AppData%\HearthstoneDeckTracker\Plugins\

📌 插件自动运行，不需要手动操作

📋 注册与报名

①

获取验证码

打开 bg_tool 或 HDT 插件，进入炉石主界面即可自动获取验证码，不需要打一局游戏。验证码显示在工具界面上，支持一键复制。

②

注册 / 登录

在网站点「登录」→ 输入完整的 BattleTag（含 #tag）+ 验证码 → 自动注册并登录。

③

报名参赛

登录后在首页点击「报名参加」→ 满 8 人自动匹配 → 正常开一局战棋 → 工具自动上传排名和积分。
